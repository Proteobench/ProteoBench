# Contact

## Email

For general questions, comments, or if none of the options below fit:
[proteobench@eubic-ms.org](mailto:proteobench@eubic-ms.org?subject=ProteoBench_query)

## GitHub Issues

Found a bug, or something not working as expected? Open an issue on
[GitHub](https://github.com/Proteobench/ProteoBench/issues).

## GitHub Discussions

For technical discussion, module proposals, or open-ended questions, use
[GitHub Discussions](https://github.com/Proteobench/ProteoBench/discussions) — this is the
preferred place for anything that benefits from public, searchable discussion (requires a GitHub
account).

## Slack

Join the ProteoBench channel on the [EuBIC-MS Slack](https://eubic-ms.org/onboarding/).

## LinkedIn

Follow [ProteoBench on LinkedIn](https://www.linkedin.com/company/proteobench) for regular updates.

## Stay updated on GitHub

"Watch" the [ProteoBench repository](https://github.com/Proteobench/ProteoBench) for email updates
on issues, discussions, and releases, or watch an individual
[results repository](https://github.com/Proteobench) for updates on a specific module.

![Screenshot showing the "watch" button on GitHub](_static/img/watch_image.png)

## Reporting a security vulnerability

Open a [GitHub issue](https://github.com/Proteobench/ProteoBench/issues) or email
[proteobench@eubic-ms.org](mailto:proteobench@eubic-ms.org?subject=ProteoBench_security). For
serious vulnerabilities, email is preferred.
